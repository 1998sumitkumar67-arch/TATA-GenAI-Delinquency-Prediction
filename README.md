# TATA iQ GenAI Powered Data Analytics Virtual Internship

## Task 2: Predicting Delinquency with AI

### Project Overview

This project was completed as part of the **TATA iQ GenAI Powered Data Analytics Virtual Internship Program**. The objective was to design a predictive modeling framework for forecasting customer delinquency using AI-assisted analytics and responsible AI principles.

---

## Business Objective

The goal was to develop an explainable predictive model that identifies high-risk customers and supports proactive intervention strategies for reducing credit delinquency.

---

## Selected Predictive Model

**Logistic Regression**

The model was selected because it provides:

- High interpretability
- Strong performance for binary classification
- Regulatory compliance support
- Transparent decision-making

---

## Key Features Used

- Credit Utilization
- Missed Payments
- Debt-to-Income Ratio
- Credit Score
- Payment History

---

## Model Evaluation Strategy

The model performance framework includes:

- Accuracy

- AUC-ROC
